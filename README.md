# HW2: RESTful API with Node.js, Express, and MongoDB

## Overview
This project builds a backend API using Node.js, Express, and MongoDB to manage Pokémon data and answer 8 analytical questions. It continues from HW1 by transforming he JSON dataset into a database and exposing endpoints.

## Setup
1. Clone the repository.
2. Run 'npm install' to install dependencies.
3. Set up MongoDB locally / use MongoDB Atlas.
4. Create a 'env.' file with MongoDB URI: MONGO_URI=mongodb+srv://MatteoMorange:student123@cluster0.vpji18r.mongodb.net/hw2db?retryWrites=true&w=majority
PORT=3000
5. Run the server using 'npm start'.
6. Run unit tests using 'npm test'.
